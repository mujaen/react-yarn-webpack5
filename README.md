# React Project Setup

Yarn berry + React v18 환경구성을 정리한 문서입니다

### 기술스택 선정

| Skill      | Version |
| :--------- | :------ |
| React      | 18.0.0  |
| Typescript | 4.6.4   |
| Yarn Berry | 3.2.0   |
| Webpack    | 5.7.2   |
| Express    | 4.18.1  |

### 상태관리

상태관리 라이브러리 후보

1. React Query
2. SWR
3. Redux ToolKit
4. Apollo(REST 지원안함)

### 테스트 도구

| Type                     | Tool                  |
| :----------------------- | :-------------------- |
| 통합 테스트(Integration) |                       |
| 단위 테스트(Unit)        | Jest                  |
| 기능 테스트(E2E)         | testing-library/react |

## Set up

1. Node는 현재 기준 LTS인 16.15.0를 사용합니다

```
nvm use 16.15.0
(npm v8.5.5)
```

2. Package Manager는 yarn(berry)를 기본으로 사용합니다

```
yarn set version berry
```

## Getting Started

```
yarn run start
```
